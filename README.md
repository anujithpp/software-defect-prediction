# Software Defect Prediction

A binary classification mini-project that predicts whether a software module contains a bug using historical software metrics from the **NASA PROMISE Repository (jm1)**.

## ⚙️ Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Download the dataset *(first time only)*
```bash
python scripts/download_dataset.py
```
Fetches jm1 ARFF from [OpenML](https://www.openml.org/d/1053), converts to CSV, saves to `data/jm1.csv`.

### 3. Run the full pipeline
```bash
python src/defect_prediction.py
```

All 6 plots are saved to `plots/` and displayed interactively.

## 🔬 Methodology

| Step | Detail |
|------|--------|
| **Dataset** | NASA PROMISE — jm1 (10,885 modules, 21 features) |
| **Target** | `defects` column — True (buggy) / False (clean) |
| **Preprocessing** | Mean imputation → StandardScaler normalisation (µ=0, σ=1) |
| **Cross-Validation** | 10-fold Stratified K-Fold on full dataset (pre-split) |
| **Split** | 80 % train / 20 % test (stratified) |
| **Models** | Random Forest · Gaussian Naive Bayes · Logistic Regression |
| **Metrics** | Accuracy, Precision, Recall, F1-Score, ROC-AUC, Confusion Matrix |
| **Imbalance fix** | SMOTE applied to training set only (never test set) |
| **Visualizations** | 6 PNG plots saved to `plots/` |

## 📊 Results

### Baseline (Before SMOTE)

| Rank | Model | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|------|-------|----------|-----------|--------|----------|---------|
| 🥇 | **Random Forest** | 80.98 % | 51.78 % | 24.23 % | **33.01 %** | **0.7174** |
| 🥈 | Gaussian Naive Bayes | 80.20 % | 47.60 % | 23.52 % | 31.48 % | 0.6758 |
| 🥉 | Logistic Regression | 80.98 % | 53.54 % | 12.59 % | 20.38 % | 0.6928 |

### After SMOTE

| Rank | Model | Accuracy | Precision | Recall | F1-Score | ROC-AUC |
|------|-------|----------|-----------|--------|----------|---------|
| 🥇 | **Logistic Regression** | 69.36 % | 33.01 % | 56.77 % | **41.75 %** | 0.6945 |
| 🥈 | Random Forest | 77.95 % | 42.38 % | 38.95 % | 40.59 % | **0.7337** |
| 🥉 | Gaussian Naive Bayes | 79.79 % | 45.74 % | 24.23 % | 31.68 % | 0.6765 |

### SMOTE Impact

| Model | ΔF1 | ΔRecall |
|-------|-----|---------|
| Logistic Regression | **+0.2136** | **+0.4418** |
| Random Forest | +0.0758 | +0.1473 |
| Gaussian Naive Bayes | +0.0020 | +0.0071 |

> **Note:** Low baseline Recall is expected — the dataset is class-imbalanced (19.3 % defective, ratio 1:4.2).
> SMOTE dramatically improves Recall (especially for LR) at the cost of Precision.
> **Random Forest** leads on ROC-AUC both before and after SMOTE.
